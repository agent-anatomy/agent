# Contributing

## Add a new agent target

1. Find the config file path the agent reads (e.g. `AGENTS.md`, `.windsurfrules`)
2. Add an entry to the `targets` array in `sync.js`
3. Update the table in `README.md`
4. Open a PR with title: `add: <agent-name> sync target`

## Improve the AGENT.md template

Found a section that should be in every project's config? Open a PR — keep it universal, not opinionated.

## Questions

Open an issue.
